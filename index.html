Análisis de datos para la optimización de información

Un sistema seguro debe ser:
  • Íntegro (con información modificable sólo por las personas autorizadas).
  • Confidencial (los datos tienen que ser legibles únicamente para los usuarios autorizados).
  • Irrefutable (el usuario no debe poder negar las acciones que realizó) y tener buena disponibilidad (debe ser estable).
Middleware

CodeEpic Middleware

La plataforma es un software que se conecta a otros sistemas y protocolos lo que sirve para:

  • Disimular una red desarticulada y distribuida.
  • Generar homogeneidad en un conjunto de aplicaciones de software.
  • Proporcionar una interfaz uniforme a los desarrolladores, para su usabilidad e interoperabilidad de sus aplicaciones.
  • Ofrecer un grupo de servicios genéricos que permite que las aplicaciones funcionen conjuntamente y evita que los sistemas dupliquen esfuerzos.
